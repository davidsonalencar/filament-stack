Filament Stack

Este repositório é gerado automaticamente a partir do processo de release do projeto principal.

Ele não deve ser utilizado para desenvolvimento direto nem receber commits manuais. Todo o conteúdo aqui presente é produzido dinamicamente por meio do pipeline de CI/CD.

Objetivo

O objetivo deste repositório é disponibilizar uma versão empacotada e distribuível do Filament Stack, contendo apenas os arquivos necessários para instalação ou distribuição.

Isso permite:
	•	📦 Distribuição simplificada do stack
	•	🔗 Links públicos para download de versões
	•	🚀 Integração com automações de deploy
	•	🧩 Separação entre repositório de desenvolvimento e artefatos de release

Como este repositório é gerado

O processo ocorre automaticamente:
	1.	Um release é criado no repositório principal.
	2.	O pipeline de CI/CD gera os arquivos de distribuição.
	3.	O conteúdo é publicado automaticamente neste repositório.

Portanto, o conteúdo aqui pode ser substituído completamente a cada nova release.

⚠️ Importante
	•	Não abra Pull Requests neste repositório.
	•	Não crie commits manuais.
	•	Não utilize este repositório para desenvolvimento.

Qualquer alteração deve ser feita no repositório principal do projeto.

Uso

Este repositório existe principalmente para:
	•	download de versões
	•	uso em automações
	•	instalação rápida do stack
	•	distribuição pública

Filament Stack

O Filament Stack é um conjunto de ferramentas e configurações para acelerar a criação de aplicações baseadas em:
	•	Laravel
	•	Filament
	•	Docker
	•	Infraestrutura moderna de desenvolvimento
